What is the rule of three?

The rule of three is one of the most useful tools in everyday math. Given any three values in a proportion, it instantly finds the fourth.

Direct proportion (A/B = C/X → X = B×C/A): used when values move in the same direction. If 1 gallon costs $4, how much do 3 gallons cost? Answer: 3×4/1 = $12.

Inverse proportion (A×B = C×X → X = A×B/C): used when values move in opposite directions. If 4 workers finish a job in 6 days, how long do 2 workers take? Answer: 4×6/2 = 12 days.

Frequently asked questions
What is the rule of three?
The rule of three is a method for solving proportion problems. Given three known values (A, B, C), it finds the fourth (X) so that the proportion holds: A/B = C/X for direct proportion, or A×B = C×X for inverse proportion.
What is the difference between direct and inverse proportion?
In direct proportion, when one quantity increases, the other increases proportionally (e.g., more items = more cost). In inverse proportion, when one increases the other decreases (e.g., more workers = fewer days to finish a job).
When do I use inverse proportion?
Use inverse proportion when two quantities move in opposite directions. Classic examples: speed vs. travel time, number of workers vs. days to complete a project, gear ratio vs. rotation speed.
Can I use this for currency conversion?
Yes! Set A = 1, B = exchange rate, C = the amount you want to convert. The result X is the converted amount. For example: 1 USD = 1.06 CAD, so 50 USD = (1.06 × 50) / 1 = 53 CAD.
