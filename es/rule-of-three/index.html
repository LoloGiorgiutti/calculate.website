¿Qué es la regla de tres?

La regla de tres es una de las herramientas más útiles en matemática cotidiana. Dados tres valores en una proporción, encuentra el cuarto al instante.

Proporción directa (A/B = C/X → X = B×C/A): se usa cuando los valores se mueven en la misma dirección. Si 1 litro cuesta $4, ¿cuánto cuestan 3 litros? Respuesta: 3×4/1 = $12.

Proporción inversa (A×B = C×X → X = A×B/C): se usa cuando los valores se mueven en sentidos opuestos. Si 4 obreros terminan un trabajo en 6 días, ¿cuánto tardan 2 obreros? Respuesta: 4×6/2 = 12 días.

Preguntas frecuentes
¿Qué es la regla de tres?
La regla de tres es un método para resolver problemas de proporción. Dados tres valores conocidos (A, B, C), encuentra el cuarto (X) para que se cumpla la proporción: A/B = C/X para la proporción directa, o A×B = C×X para la inversa.
¿Cuál es la diferencia entre proporción directa e inversa?
En la proporción directa, cuando una cantidad aumenta, la otra aumenta proporcionalmente (ej.: más artículos = más costo). En la proporción inversa, cuando una aumenta la otra disminuye (ej.: más trabajadores = menos días para terminar).
¿Cuándo uso la proporción inversa?
Usá la proporción inversa cuando dos cantidades se mueven en sentidos opuestos. Ejemplos clásicos: velocidad vs. tiempo de viaje, número de trabajadores vs. días para completar un proyecto, relación de transmisión vs. velocidad de rotación.
¿Puedo usarlo para convertir divisas?
¡Sí! Establecé A = 1, B = tipo de cambio, C = la cantidad que querés convertir. El resultado X es la cantidad convertida. Por ejemplo: 1 USD = 1,06 CAD, entonces 50 USD = (1,06 × 50) / 1 = 53 CAD.
